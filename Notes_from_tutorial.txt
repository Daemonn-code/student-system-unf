PART 3 - TEMPLATES AND htmls
    you can create variables inside the definition and pass it as arguments in render_template("html", html_variable=function_variable)
    to access variables from the python to html, is by the use of double brackets {{}}
    Loops inside the html file:
    <ul>
        {% for item in mylist %}
            <li>{{ item }} </li>
        {% endfor %}    
    </ul>
    Same case for conditional statements

You can have your base template of html {block title}

You can create template filter

You can use the function as the url in the html files
    like let's say the function is named:
        def other()

        then in the html file you can just declare:
            <a href="{{ url_for("other") }}">Click here to go to Other</a>
